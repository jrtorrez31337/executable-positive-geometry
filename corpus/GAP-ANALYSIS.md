# Corpus Gap Analysis

Verified against arXiv API on 2026-07-08. This is a convergence note for the
corpus-review pass, not a statement that the recommended PDFs have already been
filed into the local corpus.

Evidence tiers follow `corpus/README.md`:

- Established: peer-reviewed, heavily cited, or standard background.
- Toy model: rigorous and relevant, but only in idealized settings.
- Recent/speculative: young, active frontier, or not independently stabilized.

Priority tiers:

- P0: File before leaning on the claim in another lab or paper note.
- P1: File next; materially strengthens the framework.
- P2: Watchlist/frontier; cite cautiously and do not load-bear yet.

## Executive Verdict

The five gaps in the corpus-review message are real, but the holography side
needs one correction: the missing layer is not only "islands/replica
wormholes." It is the full quantum-corrected entropy chain from HRT/FLM through
QES/JLMS/entanglement-wedge reconstruction to islands. The existing corpus has
RT, ADH, and HaPPY, so it already covers the classical/QEC intuition; it does
not yet cover the semiclassical gravitational entropy technology that makes the
Page-curve story precise.

The strongest immediate additions are:

1. The sign-flip source paper, because Track B labs use that characterization.
2. Quantum corrections/QES/entanglement-wedge reconstruction, because islands
   depend on them.
3. The island/Page-curve trilogy, because it is the missing conceptual climax
   for holography, QEC, and emergent spacetime.
4. Random tensor networks, because they generalize HaPPY beyond perfect-tensor
   toy rigidity.
5. Cluster/symbol-alphabet papers, because the amplitudes side currently lacks
   its organizing combinatorics.

What is not really a gap: another generic AdS/CFT, RT, MERA, ADH, or HaPPY
introduction. Those are already represented. Complexity and SYK are gaps, but
they should be curated as a small bridge from "holographic complexity" to
"magic in SYK/wormhole models," not added as a broad SYK dump.

## Critique of the Original Five-Point List

- Point 1 is correct and is the only true "methodological violation": the labs
  are already using sign-flip technology without the source paper in corpus.
- Point 2 is correct but should be expanded upstream to HRT/FLM/QES/JLMS/DHW.
  Islands without that chain are under-explained.
- Point 3 is correct. Cluster algebras are not decorative; they organize symbol
  alphabets and singularity adjacency, so this is a structural omission.
- Point 4 is directionally correct, but should remain P2/frontier unless a lab
  actually uses cosmohedron or de Sitter wavefunction technology. The 2017
  cosmological-polytope paper is enough for the current historical bridge.
- Point 5 is correct in two separate ways: random tensor networks are a mature
  HaPPY generalization, while SYK/complexity/magic is a younger frontier. They
  should not be collapsed into one evidence tier.
